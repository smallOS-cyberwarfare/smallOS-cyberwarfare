# arjun
Find http parameters from endpoints

### Download
https://github.com/s0md3v/Arjun via pip3 install arjun --break-system-packages

### Usage  
> arjun [args]
  
### Examples   
##### Get parameters from Google Search endpoint  
```bash
arjun -u 'https://google.com/search?q=example'
```
